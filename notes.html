Functions
2/11/2016 - Gowan
We learned about basic function structure. Please review the funtion declaration keyword "def" as well as the funtion's name, "f" and the funtion's argument, "x" Understand that the "return" keyword will pass the functions return value to the rest of the code. We will expand on the consept of a function in later meetings. If you are interested in reading ahead, visit the documentation.

def f(x):
return x + 1

When this code is defined, we can call the function as shown.
x = f(2)
print(x)

When this code is run, the terminal will print 3, as shown below.
3
