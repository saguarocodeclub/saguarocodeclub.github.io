Hello World

As convention dictates, we begin learning a new language with the hello world program. The program accomplishes one thing: prints "Hello World" to the screen. Good for python, the process of doing this is very simple.

      print("Hello World")

Comments

A comment is a group of text in the code that acts as a note, or comment, to other programmers or yourself. Comments act as an explanation for complex code. There are two types of comments, one that lasts on a single line, and one that acts as a multiline comment. Single line comments are initialized by a "#", and multiline comments are initiated by "###" and ended by "###"

  # this is a single line comment

  ### this
  comment
  has
  multiple
  lines
  ###

The program's output is as follows:

    Hello World

Variables

In python, variables work in an intuitive way. To assign a variable, you define the variable name, and set it equal to its value. 

      x = 5 # now we can referance x as 5
      print(x) # prints the value of x: 5
      x = 17
      print(x) # prints the value of x: 17
The code above will output the following:

      5
      17

Basic Functions

Functions are like those in algebra. You can think of them as an input output system the input is called an argument, and the function outputs an variable. A function is defined with the keyword "def" followed by the function name and parenthesis with the argument inside. The function body begins after a colon. The body can hold something called a return value. The return value is the output of the function. 

      def f(x):
        return x + 1
You can set the return value of the function to a variable like so: 

      myVar = f(11)
      print(myVar)
The program's output is as follows: 
        12
